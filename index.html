Smart Hydroponic System

An intelligent IoT-based smart farming solution that automates plant growth through real-time environmental monitoring.

Project Overview

Designed and implemented a fully automated hydroponic system that continuously monitors water level, temperature, pH, and nutrient concentration. The system uses sensor data to trigger automated responses — pumps, alerts, and nutrient dosing — to maintain optimal plant growth conditions without manual intervention.

Key Features

  • Real-time monitoring of water level, temperature & nutrient concentration
  • Automated pump and dosing control based on sensor thresholds
  • Data logging and alert system for anomaly detection
  • Energy-efficient design suitable for small-scale farming
  • Modular sensor integration with expandability

Book Recommendation System

A Machine Learning-powered engine that delivers personalized book recommendations using collaborative filtering techniques.

Project Overview

Built a content-aware and user-based collaborative filtering recommendation engine trained on a real-world book dataset. The system analyzes user reading patterns and ratings to predict and suggest books a user is likely to enjoy — similar to how Netflix recommends shows.

Key Features

  • Collaborative filtering using user-item interaction matrix
  • Cosine similarity for finding nearest neighbors
  • Handles sparse data with matrix factorization techniques
  • Top-N book recommendations per user query
  • Data cleaning & EDA on large book datasets
